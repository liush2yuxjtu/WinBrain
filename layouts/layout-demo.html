🎨 布局系统概览

布局特性

✨ 响应式设计

所有布局组件都内置了响应式断点，在不同屏幕尺寸下自动适配。

♿ 可访问性

支持键盘导航、屏幕阅读器，符合 WCAG 2.1 AA 标准。

🎯 模块化

每个布局组件都是独立的 CSS 文件，可以按需引入。

⚡ 性能优化

使用 CSS Grid 和 Flexbox，避免 JavaScript 计算，性能更好。
